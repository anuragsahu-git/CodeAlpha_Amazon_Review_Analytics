# CodeAlpha Amazon Review Analytics

## Project Description

This project was completed as part of the CodeAlpha Data Analytics Internship. The objective of this project is to analyze Amazon customer reviews using Python and extract meaningful insights from customer feedback.

The project includes data cleaning, exploratory data analysis (EDA), data visualization, and sentiment analysis.

---

## Dataset Information

* Dataset: Amazon Reviews Dataset
* Total Records: 21,214
* Cleaned Records: 20,946
* Features: 9 columns

## Key Findings

* Most reviews in the dataset are 1-star ratings.
* The US and UK contribute the highest number of reviews.
* The average review length is around 463 characters.
* Positive and negative sentiments are nearly balanced.
* Higher ratings generally correspond to positive sentiment.
