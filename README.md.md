# Amazon Review Analytics

## Project Overview
This project analyzes Amazon customer reviews using Python.

## Objectives
- Exploratory Data Analysis (EDA)
- Data Visualization
- Sentiment Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VADER
- WordCloud

## Dataset
Amazon Reviews Dataset

## Key Insights
- Majority of reviews are 1-star.
- US and UK contribute most reviews.
- Positive and negative sentiments are nearly balanced.

## Outputs
- Rating Distribution
- Country Analysis
- Sentiment Analysis
- Heatmap
- Word Clouds
